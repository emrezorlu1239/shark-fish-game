# Shark Eats Fish

A mobile mini-game built with Expo and React Native (TypeScript). Drag the shark around the screen to eat fish and grow your score.

## Tech Stack

- [Expo](https://expo.dev) (managed workflow)
- [React Native](https://reactnative.dev) with TypeScript
- [react-native-gesture-handler](https://docs.swmansion.com/react-native-gesture-handler/) (drag)
- [react-native-reanimated](https://docs.swmansion.com/react-native-reanimated/) (60fps animations)

## How to Run

```bash
npx expo install
npx expo start
```

Scan the QR code with Expo Go (Android/iOS) or press `w` to open in a web browser.
